# Project Requirements

The following requirements make up the entire project:

- Fully functional membership platform with secure authentication
- Complete 7SHieLD holistic lifestyle discovery process
- Custom assessment system (Wayfinder + 1QRC) with automated PDF reports
- Video library with YouTube integration and podcast section
- Member "backpack" tracking progress and journey
- Email automation nurturing members through their journey
- Content management system to update content yourself
- Book sales system generating revenue from day one

The project requirements are divided into 3 phases:

## Phase 1

- Domain configuration and hosting
- Database setup
- Staging and Production environments up
- Member registration and authentication
  - Password reset functionality
  - Member profile access
  - Ability to create an anonymous username
  - Ability to export own data as csv
- Analytics collection set including data capture of:
  - name
  - email
  - address
  - phone
  - purchase history
  - last login date
  - current activity on 7Shield process
  - engagement metrics
- Data encrypted at rest
- Website core completed including routes to:
  - home
  - about
  - faq
  - member login and signup
  - legal
  - contact page
  - shop
- Book product page functional and integrated with shopify
- Email newsletter integration for mass notifications
- Administration dashboard
  - member management
  - order management
  - basic analytics
